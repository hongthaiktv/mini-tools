Text Manipulate
	Syntax:	[N][register]command
			:[range]command [register]

	i		Insert mode
	I		Insert mode at beginning of line
	A		Insert mode at end of line
	R		Replace mode
	r		replace [N]characters
	gh		select then insert
	gv		reselect previously highlighted
	v		highlight [N]characters
	V		[N]lines selection
	Ctrl-V	block selection, start with [N]characters
	Ctrl-G	switch between Select/Visual mode
	ggVG	select all
	y		copy selected
	p		[N]paste after cursor
	P		[N]paste before cursor
	d		cut/delete
	D		cut/delete to end
	c		cut/delete then insert
	C		cut/delete to end then insert
	x		delete [N]characters
	s		delete [N]characters then insert
	u		undo
	Ctrl-R	redo
	Ctrl-A	increase number by 1|[N] times
	Ctrl-X	decrease number by 1|[N] times
	{V}g+Ctrl-A		increase number by 1|[N] times in subsequence
	{V}g+Ctrl-X		decrease number by 1|[N] times in subsequence

	Line
		o		insert new line below
		O		insert new line above
		yy | Y	copy [N]lines
		:y		copy line|range
		dd		delete [N]lines
		:d		delete line|range
		d-Up|Down		delete current line and next Up|Down line
		yyp | :copy.	duplicate line
		:m{line}		move current line to "line"		Ex: :m0		move to top
		:start,end{command}		range selection			Ex: :.,$t$	copy to end
		>> | {V}> | :>		indent [N]|highlight|range lines rightwards
		<< | {V}< | :<		indent [N]|highlight|range lines leftwards
		:[range]left|center|right [width]		align line|range by column "width"
		
		Selector
			0		: top		Ex: :copy0
			^		: start		Ex: y^
			$		: bottom | end		Ex: :$y | y$
			.		: current
			*		: previous selected
			%		: all
			.-2		: previous
			.+1		: next

	Insert Mode
		Ctrl-O			run one time command
		Ctrl-R+"		paste
		Ctrl-Q+{key}	print key code
	
	Visual Mode
		U			convert to uppercase
		u			convert to lowercase
		~			invert the case
Save
	:w [file]		save current file or create new
	:wa				save all
	:x [file] | ZZ	save and exit
	:xa				save all and exit
	:wq [file]		always save and exit

Quit
	:q				exit Vim
	:q! | ZQ		discard and exit

Search (Regex support)
	/pattern		search pattern
	/pattern\c		ignore case sensitive
	/pattern\C		case sensitive
	/\<pattern\>	search whole word
	n				jump to next matched
	N				jump to previous matched
	*				search current word forwards
	#				search current word backwards
	/-arrow_up		previous search pattern
	/-arrow_down	next search pattern

	Tip: using :let @/ = "text" to set value for search register ("/)
	Bug: after set new value can using Ctrl-R+/ | "/ to obtain value but can't with arrow <Up> in search history.

Search and Replace (Regex support)
	Syntax: :[range]s/{pattern}/{string}/[flags] [count]

	:s/search_pattern/replace_string/		first occurrence in line
	:s/search_pattern/replace_string/g		all matched in line
	:%s/search_pattern/replace_string/g		all matched in file
	:*s/\(search_pattern\)/\1/						search in selected

	Replacement Pattern
		\U		: uppercase everything after
		\L		: lowercase everything after
		\u		: uppercase the first character
		\l		: lowercase the first character
		\e | \E	: stops the uppercasing or lowercasing

	Note: Must use "\(" and "\)" for grouping in regex instead of "()".

Sorting (Regex support)
	Syntax:	:[range]sort[!] [option] [/pattern/]		sort lines in all|range ASC|DESC [!]
		
		Option
			i	: ignore case
			u	: unique (group matched)
			r	: reverse pattern match (unmatch)

Navigation
	gg | Ctrl-Home | :1		move to top
	G | Ctrl-End | :$		move to bottom
	:line | line-G			go to "line" number
	f{char} | F{char}		go to next|previous "char"
	g;			[N]previous changed position
	g,			[N]next changed position
	:changes	change list

	Mark
		m{a-z} | :mark {a-z}	create current buffer mark
		m{A-Z} | :mark {A-Z}	create file mark
		m` | m' | :mark '		create previous mark
		`` | ''			jump to previous position
		`{a-zA-Z0-9}	jump to buffer|file position marked
		'{a-zA-Z0-9}	jump to buffer|file line marked
		`0 | '0			open latest file
		:marks [mark]	list all|mark
		:delm {marks}	delete marks. "marks" can be list or range		Ex: :delm a B 1 | :delm a-d	(can't delete the "'" mark)
		:delm!			delete all current buffer marks, but not A-Z and 0-9

		Note: Numbered marks (0-9) can't set directly, it auto marked when exit vim only (recent list).
	
	Tag (cross files reference)
		Ctrl-] | :tag {name}	jump to tag under cursor|name
		Ctrl-T | :pop			jump to [N]previous
		:tag		jump to [N]next
		:0tag		jump to recently
		:tags		show tag stack

Clipboard
	["{a-z}]command			overwrite register content		Ex: "ay (copy selected text to register "a")
	["{A-Z}]command			append content to register		Ex: "aP (paste content from register "a")
	:{y|d} [register]		copy|delete to "register"
	:[line]put [register]	paste from "register" after "line"
	:[line]put! [register]	paste from "register" before "line"
	:reg [register]			display all|register contents
	:set cpoptions+=>		set break line when append content to register

	Insert Mode
		Ctrl-R+{register}	paste from register

	Register Type
		""			: unnamed register (all operation)
		"{0-9}		: numbered registers (0: latest copy, 1-9: recent lines deleted)
		"{a-zA-Z}	: named registers (user define)
		"_			: black hole register (write to this register to keep other registers (unnamed) unchange)
		"=			: expression register
		".			: last inserted
		"-			: last deleted
		"/			: last search
		":			: last command
		"%			: current file name
		"#			: last file name

Window
	Syntax:	Ctrl-W+[size|order]command
			:['vert'] ['botright'] [size] command [file]
			Ex: :vert botright 5 sp

	:new | Ctrl-W+n		new horizontal window
	:vne				new vertical window
	:sp [file] | Ctrl-W+s			split horizontal
	:vs [file] | Ctrl-W+v			split vertical
	Ctrl-W+=						set all windows size equally
	Ctrl-W++ | Ctrl-W+-				increase|decrease window height by 1|size
	Ctrl-W+> | Ctrl-W+< 			increase|decrease window width by 1|size
	:res [height] | Ctrl-W+_		set window height to maximum|height
	:vert res [width] | Ctrl-W+|	set window width to maximum|width
	:set scb crb		enable scrolling synchronize
	:set noscb nocrb	disable scrolling synchronize
	:set scrollopt=ver,hor,jump		set scrolling synchronize option
	Ctrl-W+Arrow		move cursor to window by arrow direction
	Ctrl-W+w			go to next window
	Ctrl-W+W			go to previous window
	Ctrl-W+r			move window Down|Right
	Ctrl-W+R			move window Up|Left
	Ctrl-W+x			switch to next|order window
	Ctrl-W+T			move window to new tab
	Ctrl-W+K			change window to horizontal
	Ctrl-W+H			change window to vertical
	Ctrl-W+c			close current window
	Ctrl-W+o | :only	close all other windows

Tab
	:tabe [file]		open new empty|file tab
	:tabn | gt			next tab
	:tabp | gT			previous tab
	:tabc				close current tab
	:tabo				close all other tabs
	:tab command		open new tab if command create window	Ex: :tab h window

Command
	:{command}			execute "command"
	:arrow_up			previous command
	:arrow_down			next command
	:set ic				set ignore case sensitive
	:set noic			set case sensitive
	:set wrap			turn on word wrap
	:set nowrap			turn off word wrap
	:set nu				show line number
	:set nonu			hide line number
	:set {option}?		get "option" setting
	:messages			show all messages
	:Ctrl-K+{key}		print special key
	:h | :help [topic]	open help index|topic
	:set ts=4 sw=0 sts=0	set tab and indentation
	Ctrl-R+Ctrl-W		get word under cursor

	User Define
		:command[!] {Name} {command}[|{command2}..]		map|overwrite "command" to "Name" (first letter must be uppercase)	Ex: :command! CopyTop y|0put
		:command [cmd]		list all|cmd user defined commands
		:delc {cmd}			delete command "cmd" defined
		:comc				clear user defined commands

	Note:
		Commands can set as default in ~/.vimrc file (without ":"). Ex: set ic
		Using '"' at the start of a line in .vimrc file to write comment.

Scrolling
	Syntax: [N]command

	zh			scroll to left [N]characters
	zl			scroll to right [N]characters
	zH			scroll half screen left
	zL			scroll half screen right
	Ctrl-E		scroll down [N]lines
	Ctrl-Y		scroll up [N]lines
	Ctrl-D		scroll half screen down

File
	Syntax: :command [file]

	:e			refresh|open file
	:enew		new empty file
	:e #		open previous file
	:f			get file name
	Ctrl-G		get full path
	g+Ctrl-G	get statistics at cursor position
	:ol			list recent files
	:bro ol		list and open recent files
	:Lexplore [dir]		open File Explorer (Netrw)
	:filter[!] /{pattern}[\c]/ {command}	filter output of "command" by match|unmatch "pattern"

	Substitute
		# | #0	: latest close
		#Number	: open order (ASC)
		%		: current open file

		Note: Using file substitute in command to refer files recently open and close in current session.
		Ex: :sp #1 | :tabe #

Terminal
	Syntax:	:['vert'] ['botright'] ter [++option] [command]
			Ex: :botright ter ++rows=3

	:ter [command]	open terminal|run command
	Ctrl-D			close
	Ctrl-W			enter vim command
	Ctrl-W+N		text selection mode
	i				back to command mode
	Ctrl-W+"+"		paste to terminal

	Option
		++rows=height	: set height
		++cols=width	: set width

Comparison
	:diffs		open file to compare
	:difft		set current window to diff mode
	:diffp		using patch file to compare
	:diffo[!]	switch off window|all diff mode
	:diff[!]	refresh differences from internal|external changed
	[c			jump to previous
	]c			jump to next
	do | :[range]diffg [order|file]		update difference from other side
	dp | :[range]diffpu [order|file]	update difference to other side

	Note: set diffopt+=vertical to always split window vertical when enter diff mode.

Recording
	q{a-z}		record operation into register
	q{A-Z}		append operation into register
	q			stop recording
	@{reg}		execute register recorded
	@@			repeat previous execute
	:[line]@{reg}	execute register recorded in Ex mode only

	Note:
		- If using same register as copy conflict may happen.
		- @@ command run most recent executed macro, include nested run.

Spell Checking
	:spell		turn on spell checker
	:nospell	turn off spell checker
	[s			previous misspelled word
	]s			next misspelled word
	z=			suggest word under cursor
	:spellr		replace all z= word
	zG | zuG	add|undo good word
	zW | zuW	add|undo bad word

Date and Time
	strftime({"format"} [, {time}])		get string from current|time date time

	Format
		%F		ISO (1997-04-27)
		%c		local (Sun Apr 27 11:49:23 PM 1997)
		%a		day (Sun)
		%A		day (Sunday)
		%d		date (27)
		%m		month (04)
		%b		month (Apr)
		%B		month (April)
		%y		year (97)
		%Y		year (1997)
		%T		time (11:49:23)
		%H		24-hour (23)
		%I		12-hour (11)
		%M		minute (49)
		%S		second (23)
		%p		AM | PM
		%P		am | pm
		%z		UTC (+0700)
		%Z		UTC (+07)
		%x		local date (04/27/97)
		%X		local time (11:49:23 PM)

	Note: using expression register ("=) to enter function. Ex: {I}<C-R>+=strftime("%H:%M")

Shortcuts
	Syntax:	[mode]map[!] {key} {command}
			[mode]noremap[!] {key} {command}

	:map {key} {command}		map "key" to run "command"		Ex: :map mw :set wrap<CR>
	:noremap {key} {command}	map "key" to run origin "command" (no mapped command)
	:unm {key}					remove key mapped
	:mapc						clear all keys mapped. Warning: This also removes the default-mappings
	:map [key]					list all|key mapped

	:exe "normal! \<C-{Key}>"	run origin Ctrl-Key command

	MapMode
		n : Normal (map | nmap)
		v : Visual (map | vmap | xmap)
		s : Select (map | vmap | smap)
		o : Operator-pending (map | omap)
		i : Insert (map! | imap)
		c : Command-line (map! | cmap)
		t : Terminal-Job (tmap)
		l : Lang-Arg (lmap)

	Operator-pending
		Activate when press "d", "c", or "y" command in Normal mode.

		Syntax:	:omap {key} {motion}
				{Normal}[N]command[v|V|Ctrl-V][N]{key_mapped}

		Ex:	omap l i{	(select anything inside {})
			dl			(delete anything inside {})

		Common Motion
			b | e		: cursor to start|end of word
			w			: cursor to next word
			iw | aw		: current|to next word
			is | as		: current|to next sentence
			ip | ap		: current|to next paragraph
			k  | j		: previous|next line
			it | at		: inner|outer HTML|XML tag block
			i{ | a{		: inner|outer {} block
			i[ | a[		: inner|outer [] block
			i( | a(		: inner|outer () block
			i" | a"		: inner|outer "" quote
			i' | a'		: inner|outer '' quote
			i` | a`		: inner|outer `` quote
			i< | a<		: inner|outer <> block
			gN | gn		: previous|next matched search pattern
			/string<CR>	: until "string"
			T{char} | t{char}	: until previous|next "char"

	Special Keys
		<C-Key>		: Ctrl + Key    Ex: <C-A> = Ctrl-A
		<A-Key>		: Alt + Key     Ex: <A-A> = Alt-A
		<S-Key>		: Shift + Key   Ex: <S-A> = Shift-A
		<Fnum>		: F1..F12       Ex: <F1> = F1
		<CR>		: Enter
		<Tab>		: Tab
		<Esc>		: Escape
		<Space>		: Space
		<BS>		: Back Space
		<Del>		: Delete
		<Leader>	: \ (default)   Change: let mapleader = "new_key"
		<Home>		: Home
		<End>		: End
		<PageUp>	: Page Up
		<PageDown>	: Page Down
		<ScrollWheelUp>			: Scroll Up
		<ScrollWheelDown>		: Scroll Down
		<Up|Down|Left|Right>	: Arrow Up|Down|Left|Right

		Note: When running in Termux can using VolUp-Number for F1-F10 (VolUp-0).

Plugins
	coc.nvim	Console IDE - https://github.com/neoclide/coc.nvim
		~/.vimrc config:	Plug 'neoclide/coc.nvim', {'branch': 'release'}

		:CocInstall ext_name	install extension
		:CocUninstall ext_name	uninstall extension
		:CocUpdate				update extensions
		:CocList extensions		list extensions and status
		:CocList snippets		list snippets of current file type
		:CocCommand snippets.editSnippets			edit snippet for current file type in ~/.config/coc/ultisnips
		:CocCommand snippets.openSnippetFiles		choose snippet file to open by current file type

		Insert snippet
			Ctrl-J		jump to next placeholder
			Ctrl-K		jump to previous placeholder

		Example install extensions for Web Developer, Java, ShellScript, and Markdown:
			:CocInstall coc-snippets coc-html coc-css coc-tsserver coc-html-css-support coc-cssmodules coc-htmlhint coc-json coc-sql coc-java coc-clangd coc-sh coc-markdownlint

		Hint: Edit file "~/.vim/plugged/coc.nvim/plugin/coc.vim" as follow for pressing Tab|Esc button to accept|cancel code completion:
			inoremap <silent><expr> <Tab> coc#pum#visible() ? coc#pum#confirm() : "\<C-g>u\<Tab>"
			inoremap <silent><expr> <C-y> coc#pum#visible() ? coc#pum#cancel() : "\<C-y>"

		Note: Snippets extension can't open popup with multi choice placeholder if first item is unicode font (vietnamese).
